# 抹消課題アプリ（机上課題）

高次脳機能障害・注意障害のリハビリ用の抹消課題（キャンセレーション課題）アプリです。
紙のワークシート（数字抹消・ひらがな抹消）をタブレット／PCで行えるようにしました。

## 起動方法
`抹消課題アプリ.html` を**ダブルクリック**してブラウザで開くだけです。
インストール・インターネット接続は不要です（オフラインで動作）。

## iPad Pro での使い方（おすすめ）
1. このHTMLファイルをiPadに移す（メール添付／AirDrop／iCloud Drive／USB等）か、
   PCで開いているURLを**Safari**で開きます。
2. Safariで開いたら、共有ボタン → **「ホーム画面に追加」** を選ぶと、
   全画面のアプリとして起動できます（アドレスバーが消え、机上課題に集中できます）。
3. **横向き（ランドスケープ）推奨**。盤面は画面サイズに自動でフィットし、
   いちばん難しい設定でも**スクロールなしで全体が一画面に収まります**（注意・走査課題に重要）。
4. 指でも **Apple Pencil** でもタップできます。
   - ダブルタップ拡大・長押しメニュー・引っぱりスクロールは無効化済み（誤操作防止）。
   - 画面の回転にも自動で追従して再レイアウトします。
   - ノッチ／ホームインジケータ（セーフエリア）を避けて表示します。

## 3つのモード
| モード | 内容 |
|---|---|
| **数字抹消** | 0〜9 の数字の中から、指定された数字を探してタップ |
| **ひらがな抹消** | 五十音の中から、指定されたひらがなを探してタップ |
| **混在抹消** | 数字＋ひらがなが混ざった盤面。「特定の文字」「数字すべて」「ひらがなすべて」から選択（選択性注意の課題） |

## 2つのやり方
- **さがす（抹消）**：指定されたターゲットを盤面からすべて探してタップ（注意・視覚走査）
- **じゅんばん（順番）**：決まった順番にひとつずつタップ（注意の持続・系列）
  - 数字：1→2→3…（小さい順）／ …→3→2→1（大きい順）　※元の紙課題の「0→99／99→0」に相当
  - ひらがな：あ→ん（五十音順）／ ん→あ（挑戦）
  - 混在：1→あ→2→い… と数字とひらがなを交互にタップ（注意の切り替え／セットシフティング）

## 設定
- **レベル**：やさしい(8×6) / ふつう(10×8) / むずかしい(12×10)
- **ターゲット**：おまかせ（自動）または自分で選ぶ（最大3つ）
- **制限時間**：なし / 1〜3分
- **文字サイズ**：標準 / 大 / 特大（視覚に合わせて調整）
- **音フィードバック**：正解・誤反応の効果音 ON/OFF

## 操作
- ターゲットをタップ → 緑の○がつきます（正答）
- ターゲット以外をタップ → 赤い×がつきます（誤反応としてカウント）
- すべて発見、または「終了する」で結果画面へ
- **集中モード**：課題画面の「🔍 集中モード」ボタンで、ヘッダーを最小化して盤面を最大化します
  （横向きは細い左ストリップ＋プログレスバー非表示で盤面が最大に）。「✕ 解除」で戻ります。

## 結果でわかること
- **所要時間／発見数／見落とし／誤反応／正確性**
- **左右別の見落とし**：見落としが左右どちらかに偏っていないかの目安
  （半側空間無視のスクリーニングの参考）
- **見落としマップ**：盤面のどの位置を見落としたか（走査パターンの可視化）
- 結果は端末内に自動保存され、開始画面で過去の記録を確認できます
