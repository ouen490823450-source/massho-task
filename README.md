# 抹消課題アプリ（机上課題）

高次脳機能障害・注意障害のリハビリ用の **抹消課題（キャンセレーション課題）** Webアプリです。
紙のワークシート（数字抹消・ひらがな抹消）を、タブレット／PCのブラウザで行えるようにしました。
インストール不要・オフライン動作の **単一HTMLファイル**です。

🔗 **公開URL（GitHub Pages）**: https://ouen490823450-source.github.io/massho-task/

---

## 特長

### 3つのモード
| モード | 内容 |
|---|---|
| 数字抹消 | 0〜9 の数字の中から、指定された数字を探してタップ |
| ひらがな抹消 | 五十音の中から、指定されたひらがなを探してタップ |
| 混在抹消 | 数字＋ひらがなが混ざった盤面（「数字すべて」「ひらがなすべて」も選べる選択性注意の課題） |

### 2つのやり方
- **さがす（抹消）**：ターゲットを盤面からすべて探してタップ（注意・視覚走査）
- **じゅんばん（順番）**：決まった順番に1つずつタップ（注意の持続・系列）
  - 数字：1→2→3…／…→3→2→1（紙課題の「0→99 / 99→0」に相当）
  - ひらがな：あ→ん／ん→あ
  - 混在：1→あ→2→い… と数字↔ひらがなを交互（注意の切り替え／セットシフティング）

### 設定
レベル（マス数）・ターゲットの自動／手動選択・制限時間・文字サイズ（特大まで）・音フィードバック。

### 結果・分析
- 所要時間／正答（発見・順番）／見落とし／誤反応／正確性を自動集計
- **左右別の見落とし**と**見落としマップ**（半側空間無視のスクリーニングの参考）
- 結果は端末内に自動保存（過去の記録を一覧表示）

### iPad Pro 対応
- 盤面が画面サイズに自動フィット（最難設定でも**スクロールなしで全体が一画面**に収まる）
- 画面回転に自動追従、セーフエリア（ノッチ／ホームインジケータ）対応
- ダブルタップ拡大・長押しメニュー・引っぱりスクロールを無効化（誤操作防止）
- 指でも Apple Pencil でもタップ可
- Safari の「ホーム画面に追加」で全画面アプリとして起動可能
